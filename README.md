# 4S店车辆管理系统 java439

## 项目概述

本系统是一个基于Springboot+Vue的前后端分离的4S店车辆管理系统。该系统遵循MVC三层架构模式，利用Spring、SpringBoot、Mybatis、Vue和Maven等技术开发，以MySQL数据库为数据存储基础，实现了一套B/S架构的Web管理系统。系统界面简洁，操作方便。

## 技术栈

- **后端**: Spring Boot、Java EE、MyBatis
- **前端**: Vue
- **数据库**: MySQL
- **构建工具**: Maven

## 功能模块

- 车辆信息管理
- 客户信息管理
- 财务报表管理
- 库存管理

## 系统角色

- 系统管理员
- 销售人员
- 财务人员

## 运行环境

- **操作系统**: Windows/Linux/MacOS
- **数据库**: MySQL 5.7+
- **JDK版本**: JDK 1.8+
- **Node版本**: Node.js 12+

## 部署步骤

1. 环境准备：确保已安装JDK、MySQL、Node.js等依赖
2. 数据库部署：执行MySQL数据库初始化脚本
3. 后端部署：编译后端代码，运行Spring Boot应用
4. 前端部署：编译前端代码，部署到Web服务器

## 目录结构

```
├── backend                # 后端代码目录
│   ├── src
│   ├── pom.xml
├── frontend               # 前端代码目录
│   ├── src
│   ├── package.json
└── sql                    # SQL脚本目录
    └── initialize.sql      # 数据库初始化脚本
```

## 核心亮点

- **前后端分离**: 前后端分离的设计使得项目更加便于维护和扩展。
- **响应式界面**: 界面简洁，支持多种设备访问。
- **灵活的系统架构**: 基于MVC模式设计，有利于系统分层开发与维护。
- **完整的权限控制**: 系统角色明确，权限控制精准。
